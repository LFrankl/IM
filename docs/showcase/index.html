一切你需要的
即时通讯功能

从私聊到群组，从好友管理到个人空间，完整还原 QQ 的核心使用场景。

💬
实时私聊
基于 WebSocket 的实时消息传输，支持文字、图片、文件发送，消息历史分页加载。
👥
群组聊天
创建、搜索、加入群组，实时群消息，群主管理成员、踢人、设置邀请权限。
❤️
好友系统
搜索用户、发送好友请求、接受/拒绝、设置备注，完整的社交关系链管理。
🏠
个人空间
类 QQ 空间的动态发布、点赞与评论，支持图文混排，好友动态聚合展示。
🖼️
图片 / 文件传输
聊天内上传图片和文件，服务端存储，消息中内联渲染图片预览，文件可下载。
😊
Emoji 选择器
内置 5 分类 200+ Emoji，光标位置精准插入，支持连续快速多选，体验流畅。
